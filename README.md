# Дубкова Вероника Александровна 4ИСИП-422

## Приложение для учета расходов пользователей

---

## Описание проекта

Данное приложение реализовано на платформе **WPF (.NET Framework)** с использованием **Entity Framework** для взаимодействия с базой данных **SQL Server**.  
Оно предназначено для **авторизации, регистрации пользователей, ведения категорий и учета платежей**.  
В приложении реализовано **разделение ролей** - разграничение прав доступа для администратора и обычного пользователя.

---

## Основной функционал

* Авторизация и регистрация пользователей
  Пользователи могут создавать учtтные записи. Пароли хранятся в базе данных.  

* **Для администратора:** 
  - добавление, редактирование и удаление пользователей, категорий и платежей;
  - формирование отчетов в виде диаграмм с возможностью экспорта в Word и Excel.

* **Для пользователя:**
  - просмотр списка всех пользователей с фотографиями их профиля с возможностью поиска и фильтрации по ФИО.

* **Интуитивный интерфейс** 
  Приложение выполнено с визуальным разделением страниц, подсказками в полях ввода и плавной навигацией между страницами.
